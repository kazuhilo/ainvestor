# 今後の方針について

- Status: Proposed
- Date: 2026-05-19
- Deciders: kazuhiro, toshiki

## Context

トレーディング戦略をどのアプローチで開発・検証していくかの方針を定める必要がある。戦略の発想方法は多数存在する（テクニカル分析、ML、ファンダメンタル、イベントドリブン等）が、開発リソースを集中させるために起点となるアプローチを 1 つ選ぶ。また、最初から独自ルールを盛り込むのではなく、まず基本的な仕組みを動かしてからカスタマイズしていく開発スタイルを採用する。

## Decision

**イベントドリブン戦略の検証フロー**をベースに開発・検証していく。

1. ニュース＋株価データを 5 年分集める（Finnhub や Alpaca News）
2. 「決算発表後の翌日」「ガイダンス上方修正」など明確なイベントで分類
3. イベント後 N 日間のリターンを統計的に集計
4. 統計的に有意な傾向だけ採用（多くは有意でない）
5. 取引コスト・スリッページ込みでバックテスト
6. 紙トレで 6 ヶ月、機能したら実弾少額

まずはこのフローに沿った**基本的な仕組みを実装してトレードできる状態**を作るところからスタートし、そこからカスタマイズして独自の取引ルールを加えていく。

## Consequences

### Positive

- 小さく始めて検証しながら拡張できる
- 小さく進めながら株・市場・イベント影響についての知識を実践を通じて広げていける

### Negative

- なし
